Uniform Resource Locator(URL)

Every website has a URL. It is a unique identifier used to locate resources on the web(images,sound files, and hypertext pages)
Displayed below is the structure every URL takes.

  • The Protocol specifies how information from the link is transferred.
  • The domain name identifies the server who is hosting the web page
  • The domain name has to be registered with a domain name registrar which cost $$
  • The Path will usually refer to a file or folder (directory) on the webserver
