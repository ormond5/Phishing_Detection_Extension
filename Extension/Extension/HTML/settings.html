Phishing Detector

Phishing sites sometimes contain IP address in the URL.

Phishing URLs are longer in the hopes to confuse an individual

Phishers can shorten their URL is they use a URL shortening service.

>Using “@” symbol in the URL leads the browser to ignore everything preceding the “@” symbol and the real address often follows.

The presence of '//' in the URL path means that the user will be redirected to another website.

Phishers tend to add prefixes or suffixes separated by (-) to the domain name so that users feel that they are dealing with a legitimate webpage

Multiple dots in the domain name are a sign of Phishing!

I check to see if the website has HTTPS and has a trusted certificate greater than two years.

Favicon is a icon associated with a specific webpage. If I find that the favicon is loaded from a domain other than that shown in the address bar,then the webpage is likely to be considered a Phishing attempt.

When computers talk to each other they send information through ports. When accessing a webpage, there should only be one port number open for communication.

The phishers may add the “HTTPS” token to the domain part of a URL in order to trick users.

In legitimate webpages, the webpage address and most of objects embedded within the webpage are sharing the same domain.

in HTML code an a tag means that u are redirecting to whatever follows it.

It is common for legitimate websites to use 'Meta' tags to offer metadata about the HTML document;

A phisher might redirect the user’s information to his personal email

The details of this are confusing. Just trust me!

IFrame is an HTML tag used to display an additional webpage into one that is currently shown, usually hidden.
